 # 📊 Sales Data Analysis Project

## 📌 Problem Statement

The goal of this project is to analyze sales data by combining multiple datasets and extracting meaningful business insights such as sales trends, top-performing products, and regional performance.

---

## 📁 Dataset

This project uses multiple datasets:

* Sales Data (transactions, quantity, total sales)
* Products Data (product details, category, price)
* Customers Data (customer information)
* Stores Data (store and regional data)

---

## 🛠️ Tools & Technologies Used

* Python 🐍
* Pandas
* Matplotlib
s
---

## 🔄 Data Processing

* Loaded multiple CSV files using Pandas
* Merged datasets using:

  * ProductID
  * CustomerID
  * StoreID
* Created a final structured dataset for analysis

---

## 📊 Analysis & Visualizations

### 📈 1. Monthly Sales Trend

* Extracted month from date column
* Calculated monthly total sales
* Visualized using line chart

💡 **Insight:**

* Sales fluctuate over time
* Some months show higher peaks while others show dips

---

### 🏆 2. Top 5 Products by Sales

* Grouped data by Product Name
* Calculated total sales
* Selected top 5 products

💡 **Insight:**

* Laptop and TV generate highest revenue
* Camera and Smartphone contribute moderately
* Gaming Console has relatively lower sales

---

### 🌍 3. Sales by Region

* Grouped data by region
* Calculated total sales per region
* Visualized using bar chart

💡 **Insight:**

* North region generates highest sales
* South is second highest
* East and West regions have lower performance

---

## 🚀 Conclusion

This project demonstrates how data analysis can uncover valuable business insights such as:

* Sales trends over time
* High-performing products
* Strong and weak regions

These insights can help businesses make data-driven decisions.
