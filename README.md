 # 📊 Sales Data Analysis Project

 ## 💡 About This Project

This project demonstrates my data analysis skills using Python, including data cleaning, data merging, and visualization.

---------

## 📌 Problem Statement

The goal of this project is to analyze sales data by combining multiple datasets and extracting meaningful business insights such as sales trends, top-performing products, and regional performance.

---

## 📁 Dataset

This project uses multiple datasets:

* Sales Data (transactions, quantity, total sales)
* Products Data (product details, category, price)
* Customers Data (customer information)
* Stores Data (store and regional data)

---

## 🛠️ Tools & Technologies Used

* Python 🐍
* Pandas
* Matplotlib
s
---

## 🔄 Data Processing

* Loaded multiple CSV files using Pandas
* Merged datasets using:

  * ProductID
  * CustomerID
  * StoreID
* Created a final structured dataset for analysis

---

## 📊 Analysis & Visualizations

### 📈 1. Monthly Sales Trend

* Extracted month from date column
* Calculated monthly total sales
* Visualized using line chart

💡 **Insight:**

* Sales fluctuate over time
* Some months show higher peaks while others show dips

---

### 🏆 2. Top 5 Products by Sales

* Grouped data by Product Name
* Calculated total sales
* Selected top 5 products

💡 **Insight:**

* Laptop and TV generate highest revenue
* Camera and Smartphone contribute moderately
* Gaming Console has relatively lower sales

---

### 🌍 3. Sales by Region

* Grouped data by region
* Calculated total sales per region
* Visualized using bar chart

💡 **Insight:**

* North region generates highest sales
* South is second highest
* East and West regions have lower performance

-----


## 📊 High-Value Customers Analysis
# 🔍 Objective

Identify top customers based on total spending.

# 🧠 Approach
* Group data by CustomerName
* Calculate total purchase amount
* Sort in descending order
* Select top customers

----- 

## 📈 Visualization

Bar chart showing top 5 customers based on total spending.

# 💡 Key Insights
* Eva is the highest spending customer

------

# 📌 Business Insight

## 🏬 Store Performance Analysis

Analyze which stores are generating the highest revenue.

Bar chart comparing store performance.

## 🧩 Category-wise Sales Distribution
# 🔍 Objective

Understand which product categories contribute most to total sales.

# 🧠 Approach
* Group data by Category
* Calculate total sales
* Visualize percentage contribution

------

## 📈 Visualization

Pie chart showing category-wise distribution.

# 💡 Key Insights
* Electronics dominates (~77%) of total sales
* Gaming (~10%) is second highest
* Other categories (Office, Accessories, Networking) contribute less

-------

# 📌 Business Insight
* Business is heavily dependent on Electronics
* Opportunity to grow other categories
* Diversification can reduce risk
